RenCMS is a modular Content Management System (CMS) built for developers and content creators.
It provides an easy way to manage e-books, articles, albums, galleries, and products, with role-based access and secure authentication.

Key points:

CRUD management for roles, users, articles, albums, categories, galleries, and products

Built-in search & filtering for content

OAuth login and reCaptcha integration for secure authentication

Focused on user security and data integrity
